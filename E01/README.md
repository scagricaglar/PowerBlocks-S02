# PowerBlock s02e01 - Obje, JSON ve YAML

Bu bölümüde:
- Pek çok modern otomasyon sürecinde ve aracında yaygın olan ‘obje’lerin ne olduğu, neden önemli olduğu ve nasıl kullanacağı ele alınmış,
